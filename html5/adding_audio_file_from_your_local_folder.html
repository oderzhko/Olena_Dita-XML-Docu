How to add an audio file from your local folder

You can add two types of audio files to your PowerPoint slide: an audio file from your local folder or a recording made directly in the PowerPoint.

Make sure you have created the PowerPoint slide.
Here you will find description of adding audio file from your local folder.
  1. In your newly created PowerPoint slide go to Insert tab > Media.
  2. Click Audio button.
  3. Select Audio on My PC... option.
  4. Choose the required audio file from the local folder.
  5. Click Insert button.
The audio file is added to your PowerPoint slide.
