How to create your PowerPoint presentation using a template

Here you will find a description of how to create a PowerPoint presentation using an available template.

Make sure you have the PowerPoint app installed on your PC.

  1. Go to a task bar and select PowerPoint app.
  2. Once the PowerPoint app is open, go to File > New.
  3. Select one of the presentations available in the Office or Customtabs.
  4. You can also search for a specific template or theme in the search bar using a specific keyword.
  5. As you selected the required theme, click it and then click the Create button.
Your have now created your PowerPoint presentation with one of the available templates.
